# CyberSafetyBot (Chatty)

CyberSafetyBot is a Windows Forms application built in C# that helps users learn about basic cyber safety concepts through an interactive chatbot and quiz system.

## Features

### Chatbot (Chatty)
- Responds to user input using keyword-based responses
- Covers cyber safety topics such as:
  - Phishing
  - Malware
  - Password safety
  - Scams
  - Privacy
  - Two-factor authentication (2FA)
  - Online safety tips
- Includes tone selection (Friendly, Serious, Professional)
- Maintains simple memory of topics discussed during the session

### Quiz System
- Multiple-choice cyber safety quiz
- Tracks user score
- Displays feedback for correct and incorrect answers
- Shows final score at the end of the quiz

### User Interface
- Built using Windows Forms (WinForms)
- Interactive buttons, text inputs, labels, and radio buttons
- Activity log to track user interactions

## Technologies Used
- C#
- .NET Windows Forms (WinForms)

## How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the application (Start button or F5)

## Purpose
This project was created as part of a learning exercise to demonstrate:
- Basic chatbot logic using conditional statements
- Event-driven programming in WinForms
- Simple user interaction and quiz handling
- Basic memory storage using lists
